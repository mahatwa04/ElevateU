ElevateU Frontend (Next.js 14 + TypeScript)

Quick start

1. cd frontend
2. npm install
3. cp .env.example .env.local and set NEXT_PUBLIC_API_BASE
4. npm run dev

What this scaffold includes
- Next.js 14 app router
- TypeScript
- TailwindCSS
- React Query setup
- Axios instance with JWT access+refresh interceptors
- Zod + React Hook Form forms
- Starter pages: login, register, feed, profile, achievements, leaderboard
- Starter components and hooks

Notes
- Install ShadCN UI and configure if you want their components. This scaffold uses simple Tailwind components to stay dependency-light.
