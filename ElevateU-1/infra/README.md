# Infra

This directory contains the infrastructure-related files for the ElevateU project.

## Docker Setup

- **docker-compose.yml**: Defines the services and configurations for Docker containers used in the project.
- **Dockerfile.web**: Instructions for building the web service Docker image.
- **Dockerfile.frontend**: Instructions for building the frontend Docker image.
