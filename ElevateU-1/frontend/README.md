# ElevateU Frontend

Welcome to the ElevateU frontend project! This project is designed to provide a user-friendly interface for university students to showcase their achievements and engage with their peers.

## Overview

ElevateU is a platform where students can post their achievements in various fields such as academics, sports, music, and more. Users can like, comment, and follow each other, fostering a community of encouragement and recognition. The platform also features rankings and leaderboards based on user interactions.

## Features

- **User Authentication**: Signup and login functionality using campus email.
- **Achievement Posting**: Users can post achievements with photos or videos.
- **Engagement**: Like, comment, and follow other users.
- **Rankings**: Field-based rankings to recognize top achievers.
- **Leaderboards**: Display weekly, monthly, and all-time leaderboards.
- **Skill Tags & Endorsements**: Users can tag their skills and endorse others.
- **Event & Competition Integration**: Participate in events and competitions.

## Tech Stack

- **Frontend Framework**: React (with Vite)
- **State Management**: Context API or Redux (to be determined)
- **Styling**: CSS Modules or styled-components (to be determined)
- **API Integration**: Axios or Fetch API for making requests to the backend.

## Getting Started

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/ElevateU.git
   cd ElevateU/frontend
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. **Open in Browser**: Navigate to `http://localhost:3000` to view the application.
